Forja de Currículos - Guia Completo de Features

1. Sistema de Autenticação e Login

Login Local (sem autenticação externa)
O sistema mantém todos os dados do usuário exclusivamente no localStorage do navegador. Nenhuma informação é enviada a servidores externos.
// Dados armazenados em localStorage { "user": { "fullName": "João Silva", "birthDate": "1990-01-15", "email": "joao@example.com", "phone": "(11) 98765-4321", "address": "Rua A, 123, São Paulo", "createdAt": "2026-02-06T22:45:00.000Z" } }
Toca dos Hobbits - Design da Tela de Login
A tela de login é inspirada na toca dos Hobbits com:
  • Decoração visual em forma de portal redondo
  • Gradiente de cores medievais
  • Animação de fundo com padrão de diamantes
  • Bordas douradas ornamentadas
Easter Eggs
A tela de login contém surpresas para usuários atentos:
  1. Clique na frase "Um anel para governar..."
    Ativa uma sequência de 5 diálogos dos anéis do Senhor dos Anéis
  2. Konami Code: ↑ ↑ ↓ ↓ ← → ← → B A
    Ativa o "Modo Escuro do Anel" com efeito visual especial

3. Sistema de Formação Acadêmica

Cálculos Automáticos para Cursos
O sistema calcula automaticamente informações sobre cursos não finalizados:
  • Previsão de Término: Calculada a partir da data de início + duração
  • Tempo Restante: Em meses, até a previsão de término
  • Porcentagem Concluída: Com barra visual de progresso
Exemplo de Cálculo
Entrada: - Curso: Engenharia de Software - Instituição: Universidade XYZ - Início: Fevereiro/2022 - Duração: 48 meses Cálculo: - Data de término: Fevereiro/2026 - Se hoje é: Fevereiro/2026 - Meses restantes: 0 (Concluído) - Progresso: 100%
Exibição no Currículo
- Se concluído: Mostra apenas "Concluído"
- Se em andamento: Mostra "Previsão de término: Mês/Ano | X meses restantes | Y% concluído"
- No site: Todos os cálculos detalhados são visíveis
- No PDF: Apenas previsão de término aparece

4. Experiência Profissional

Campos de Experiência
Cada entrada de experiência contém:
  • Cargo/Posição
  • Nome da Empresa
  • Mês/Ano de Entrada
  • Mês/Ano de Saída (ou "Atual")
  • Descrição de Atividades (opcional)
Marcação de Trabalho Atual
Ao marcar "Trabalho atual", o campo de data de saída é automaticamente desabilitado e o currículo exibe "Atual" em vez da data.

5. Sistema de Skills (Habilidades)

Lista Suspensa com 40+ Sugestões
O sistema contém sugestões pré-carregadas de skills:
LINGUAGENS: JavaScript, TypeScript, Python, Java, C, C++, PHP, Go, Rust WEB: HTML, CSS, React, Angular, Vue, Node.js, Express BANCO DE DADOS: MySQL, PostgreSQL, MongoDB, Redis FERRAMENTAS: GitHub, Docker, Kubernetes, Power BI, Excel SOFT SKILLS: Proatividade, Colaboração, Comunicação, Resolução de problemas, Liderança, Criatividade
Escrita Livre de Skills
Além das sugestões, o usuário pode digitar qualquer skill personalizada que não esteja na lista pré-carregada.

6. Sistema de Links Opcionais

Links Suportados
  • GitHub - Perfil do desenvolvedor
  • LinkedIn - Rede profissional
  • Portfólio - Site pessoal ou portfólio
Todos os links são completamente opcionais. Se não preenchidos, não aparecem no currículo.

7. Sistema de Certificados

Upload e Armazenamento
Certificados podem ser em formato:
  • PDF
  • JPEG/JPG
  • PNG
  • GIF
Armazenamento Base64
Os certificados são convertidos para base64 e salvos no localStorage, permitindo que sejam visualizados e baixados sem acesso ao arquivo original.
Funcionalidades de Certificados
  • Visualizar arquivo original
  • Fazer download do certificado
  • Deletar certificado do sistema
  • Link para página de certificados no currículo

8. Editor de Currículo

Personalização de Cores
  • Cor de Fundo: Define fundo do currículo
  • Cor do Texto: Define cor principal do texto
  • Cor de Destaque: Define cor dos títulos e bordas
Ajuste de Tamanho
Slider para ajustar o tamanho da fonte entre 12px e 24px.
Visualização em Tempo Real
Todas as mudanças são aplicadas instantaneamente ao currículo exibido, permitindo que o usuário veja o resultado antes de exportar.
Resetar Padrões
Botão para voltar às cores e tamanhos padrão (tema medieval com ouro).

9. Visualização em Tempo Real do Currículo

Currículo Sempre Visível
O currículo é exibido em tempo real à medida que o usuário preenche os dados:
  • Digitando o nome → atualiza imediatamente no currículo
  • Adicionando experiência → aparece na seção correspondente
  • Preenchendo formação → mostra previsão de término ou status
  • Adicionando skills → atualiza lista de habilidades
Estrutura Padrão do Currículo
[CABEÇALHO] - Nome - Título Profissional - Contato (Email, Telefone, Endereço, Nascimento) [SEÇÕES] - Sobre Mim - Experiência Profissional - Formação Acadêmica - Habilidades - Links - Certificados

10. Exportação em PDF

Download Automático
Ao clicar em "Exportar PDF", o currículo é baixado com:
  • Nome do arquivo: curriculo_[nome do usuário].pdf
  • Todas as personalizações de cores e tamanho
  • Formatting preservado
  • Formato A4 em retrato
Nota sobre Exportação
O botão "Exportar PDF" fica desabilitado até que o usuário tenha pelo menos uma entrada de educação ou experiência cadastrada.

Instruções Complementares

Para Usuários Novos
  1. Acesse index.html
  2. Preencha seus dados de cadastro ou clique "Pular Tudo"
  3. Comece preenchendo as abas na ordem desejada
  4. Visualize as mudanças em tempo real no currículo
  5. Quando satisfeito, exporte para PDF
Para Usuários Retornantes
  1. Os dados são automaticamente carregados do localStorage
  2. Você pode editar qualquer seção
  3. As alterações são salvas automaticamente
  4. Exporte PDF quando quiser uma cópia atualizada
