SUPORTE

Problemas Comuns:

"Dados não salvam"
  → localStorage pode estar desabilitado
  → Modo incógnito não funciona bem
  → Tente outro navegador

"PDF não exporta"
  → Recarregue a página
  → Tente com menos dados
  → Verifique console (F12)

"Cores incorretas"
  → Clique "Resetar para Padrão"
  → Limpe cache do navegador

"Certificado não sobe"
  → Limite: PDF/Imagens apenas
  → Preencha o nome
  → Arquivo não muito grande
