Estrutura de Dados localStorage

O sistema usa 9 chaves no localStorage para persistência completa: 

• user - Dados pessoais (nome, email, telefone, endereço, etc)
• education - Array de cursos com cálculos automáticos
• experience - Array de experiências profissionais
• skills - Array de habilidades (sugeridas + personalizadas)
• links - GitHub, LinkedIn, Portfólio (opcionais)
• certificates - Array de certificados em base64
• editorSettings - Cores e tamanho de fonte
• lastAccess - Data/hora do último acesso
• lastCurriculumUpdate - Data/hora da última atualização
